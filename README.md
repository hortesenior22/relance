Relance — Frontend

Relance es una plataforma web diseñada para conectar estudiantes, empresas y centros educativos, facilitando la gestión de prácticas y el acceso al primer empleo.

Tecnologías
React 18 — Librería para la construcción de interfaces
Vite — Entorno de desarrollo rápido y herramienta de build
Tailwind CSS — Framework de estilos basado en utilidades
Supabase (JS v2) — Backend como servicio (autenticación y base de datos)
React Router v6 — Gestión de rutas en la aplicación
